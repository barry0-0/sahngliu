# S2B2C 供应链系统 PRD

## 一、项目概述
本系统为 S2B2C 大宗商品供应链交易平台，包含运营端、商家端、商城PC、H5小程序四端。

## 二、用户体系
- 统一用户管理，通过 userType 区分会员/商家
- 商家通过审核入驻机制获得商家权限

## 三、产品边界

## 八、核心流程

### 入驻流程
注册 → 申请入驻 → 提交资质 → 运营审核 → 通过获得商家权限

### 现货交易
下单 → 合同签署 → 线下付款 → 填写物流单号 → 确认收货 → 结算

### 竞价交易
发布竞价 → 审核通过 → 买家报名 → 竞价出价 → 中标 → 合同签署 → 结算

### 询价沟通
买家询价 → 商家报价 → 双方沟通 → 下单/结束
