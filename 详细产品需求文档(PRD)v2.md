# 采销云 S2B2C 供应链交易系统 - 详细产品需求文档 (PRD)

## 一、系统概述
本项目为“采销云简易版 S2B2C 供应链系统”，服务于大宗商品/供应链业务的数字化交易。本系统涵盖四大核心端口：**运营端 (PC)**、**商家端 (PC)**、**采购商城 (PC)**、**移动端 (H5小程序)**。

**核心约束与裁剪**：
1. 取消独立的“供应商端”，缩减采购链路。
2. 取消所有的“保证金”与“营销中心”逻辑。
3. 砍掉复杂的仓储调度，物流仅保留三方快递/运单号登记。
4. 全端采用单一用户体系架构，商家通过商城注册并审核后开通商家权限。

---

## 二、端口模块详解

### 1. 运营后台端 (Admin PC)
服务对象：平台运营管理员，负责全平台的审核、监管与资金对账。

#### 1.1 业务看板 (Dashboard)
- **展示内容**：待审核入驻数、待处理发票数、平台累计流水、抽成收益汇总。

#### 1.2 会员与入驻审核中心
- **菜单功能**：商家入驻审核列表、全平台用户列表。
- **筛查功能**：按“公司名称”、“手机号”、“状态（待审/已过/驳回）”进行搜索。
- **数据字段**：用户ID、公司抬头、统一社会信用代码、注册手机号、申请时间、当前状态。
- **操作按钮**：
  - `[查看资质]`：进入二级页面查看企业执照扫描件。
  - `[审核通过]` / `[驳回重填]`：在二级页面进行资质审批流转。

#### 1.3 订单与资金监管
- **菜单功能**：全平台订单监控。
- **筛查功能**：按“订单编号”、“商家名称”、“买家名称”、“订单状态”过滤。
- **数据字段**：订单号、所属商家、买家抬头、商品信息、交易金额、支付状态、物流状态。
- **操作按钮**：`[查看详情]`

#### 1.4 结算与返佣对账
- **菜单功能**：平台抽成对账单、发票开具申请管理。
- **筛查功能**：按“账期月份”、“商家名称”、“核销状态”筛选。
- **数据字段**：账单编号、关联订单、总货值、应扣服务费(佣金)、打款凭证照片、核销状态。
- **操作按钮**：
  - `[查看水单]`：预览商家上传的佣金汇款凭证。
  - `[确认核销]`：财务确认款项到账，更改为已清算状态。
  - `[发票登记]`：给商家登记平台开具的服务费发票号码。

### 2. 商家工作台 (Merchant PC)
服务对象：通过审核的入驻商家，在此进行商品挂牌、竞价发起、发货及对账。
*(注：去除独立登录页面，由商城入口统一鉴权进入。去除独立物流运费模板配置，仅保留发货填单功能)*

#### 2.1 经营看板与店铺
- **数据模块**：待发货订单数、待对账单数、待缴纳佣金总额。
- **店铺配置**：
  - 表单字段：商铺招牌名称、店铺行情简介公告。
  - 按钮：`[保存资料]`。

#### 2.2 商品中心
- **菜单功能**：大豆现货挂牌商品管理、一客一价专属策略。
- **列表字段**：大豆名称、价格、余量库存、扣点佣金比例、商品状态。
- **操作按钮**：`[+ 发布挂牌商品]` (跳转二级页面)、`[一客一价特惠绑定]` (弹窗)。
- **发布挂牌表单 (二级页面)**：
  - 输入字段：商品名称、挂牌价格(元/吨)、供应库存(吨)、交易质量参数(文本:水分/蛋白等)。
  - 按钮：`[提报上架]`。
- **一客一价表单 (弹窗)**：
  - 下拉选框：选择买家公司、绑定挂牌商品。
  - 输入字段：折扣单价(元/吨)。

#### 2.3 订单与沟通
- **菜单功能**：销售订单与发货、即时报价单留言。
- **订单列表字段**：订单号、买家抬头、商品明细、结算总金额、进度状态、物流发运。
- **搜索筛查**：支持输入单号或买家名称模糊匹配。
- **发货操作 (二级页面)**：
  - 输入字段：分派物流承运车队名称、装运车牌号码/三方快递单号。
  - 按钮：`[确认派车发运]`。
- **即时沟通**：
  - 左侧：有过对话的采购商列表。
  - 右侧：聊天记录框，底部输入框及`[发送回复]`按钮。

#### 2.4 竞拍专场中心
- **操作按钮**：`[+ 提报竞买场次]`。
- **表单字段**：竞拍公告主题、起拍基准价、最低加价阶梯、截止竞价时间。

#### 2.5 账目与发票
- **操作流程**：查看平台出具的结算账单 -> 上传支付佣金的水单 -> 平台核销。

### 3. PC 采购商城 (Mall PC)
服务对象：全网采购商（买家），未登录可浏览，登录可交易。

#### 3.1 商城首页与供应大厅
- **模块**：行情焦点图、最新现货挂牌列表。
- **筛查功能**：搜索框（按品类/商家搜索）。
- **商品卡片展示**：商品名称、所属商家、价格、库存、简要参数。
- **操作按钮**：`[查看详情]`、`[加入采购清单]`、`[立即买入]`。

#### 3.2 竞价大厅
- **模块**：展示正在竞拍和即将开始的专场。
- **列表字段**：专场名称、发起商家、起拍底价、加价幅度、倒计时。
- **操作按钮**：`[进入竞拍大厅]` (二级页面，提供出价输入框和出价记录)。

#### 3.3 产销对接（咨询室）
- **功能**：与商家的在线客服会话功能。买家针对某商品发起询价，商家后台实时接收并回复。

#### 3.4 会员中心
- **买家订单**：查看已买到的商品。
- **物流查询**：在订单详情中查看商家填写的“承运车队”及“车牌号/快递单号”。（原复杂物流大屏功能移除）。
- **认证入驻**：普通用户提交企业执照申请成为“商家”。
  - 表单字段：企业名称、信用代码、法人姓名、营业执照图片。

---

### 4. 移动端 H5 小程序 (H5)
服务对象：移动端买家及便携式业务跟进。功能与 PC 商城对齐，采用移动端交互规范（底部导航栏，单列/双列瀑布流）。

#### 4.1 底部导航栏结构
- **首页 (现货大厅)**：搜索框、轮播图、现货列表。支持点击进入商品详情。
- **竞拍场**：展示所有竞价专场卡片，卡片包含倒计时。
- **消息 (产销对接)**：与商家的历史会话列表，点击进入聊天页面。
- **我的 (会员中心)**：
  - 顶部个人信息。
  - 订单中心（全部、待付款、待发货、待收货）。
  - 功能网格：商家入驻申请、收货地址管理。
