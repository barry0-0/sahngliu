# 《The Mom Test》提炼存档

原著：Rob Fitzpatrick,《The Mom Test: How to Talk to Customers & Learn if Your Business is a Good Idea When Everyone is Lying to You》(2013)。

提炼四原则遵循：忠于原著 / 可执行 / 可追溯（标章节）/ 有边界。

## 章节 → 产出映射
- 前言 & 第1章「The Mom Test」→ 三条铁律（谈生活不谈点子 / 问具体过去 / 多听少说）；坏问题→好问题改写规则。
- 第2章「Avoiding Bad Data」→ fluff 三形态（泛泛观点/未来保证/假设称赞）；赞美是危险假信号。
- 第3章 挖掘真实痛点 → "真痛点必有过去代价"；谈过去而非未来框架。
- 第4章 不要 pitch → 反模式"把访谈当路演"。
- 第5章「Commitment & Advancement」→ 承诺与推进框架（时间/声誉/金钱三种代价）。
- 第6-7章 找对人、组织与记录 → 主动问最怕的答案；留可核对事实。

## 收录判断（能否改写成 AI 可执行指令）
- 可执行 → 收录：三铁律、好问题改写、fluff 过滤、承诺分级、信号清单。
- 不可执行/背景 → 降级：作者创业故事案例仅作注释，不进规则。

## 边界
- 2013 出版，方法稳定；仅解决定性取真信号，不解决定量/规模/可用性测试。
- 单次样本小，需多访谈交叉验证。
- 与套件分工：Mom Test = 逐句访谈话术；Torres = 发现节奏与机会结构；Story Mapping = 需求结构化。

（本存档基于原著框架与公认章节结构提炼；如个别规则对应章节按二手细节有出入，以原书为准。）
